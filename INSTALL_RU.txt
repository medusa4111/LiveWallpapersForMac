Live Wallpapers for Mac — установка

Системные требования: macOS 13 или новее.

1. Откройте файл Live Wallpapers for Mac-<версия>.dmg.
2. Перетащите Live Wallpapers for Mac.app на ярлык Applications.
3. Если установлена старая версия, подтвердите её замену.
4. Откройте папку «Программы» и запустите Live Wallpapers for Mac.
5. При первом запуске macOS может заблокировать приложение, потому что оно пока не нотарифицировано Apple. Нажмите по приложению правой кнопкой мыши, выберите «Открыть», затем ещё раз «Открыть». Если кнопки нет, откройте «Системные настройки» → «Конфиденциальность и безопасность» и нажмите «Всё равно открыть».
6. Разрешите запрошенный доступ в настройках macOS. После запуска значок приложения появится в строке меню; значка в Dock не будет.

Обновление

Для обновления откройте новый DMG и снова перетащите приложение в Applications с заменой старой версии. Не переименовывайте приложение и не переносите его в другую папку. Постоянные имя, путь, Bundle ID и подпись позволяют macOS распознавать обновление как то же приложение и сохранять выданные разрешения.
