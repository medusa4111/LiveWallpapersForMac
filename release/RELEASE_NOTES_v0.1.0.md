# Live Wallpapers for Mac 0.1.0

Первый подписанный релиз Live Wallpapers for Mac.

## Установка на русском

1. В разделе **Assets** ниже скачайте файл `Live.Wallpapers.for.Mac-0.1.0.dmg`.
2. Откройте DMG и перетащите `Live Wallpapers for Mac.app` на ярлык `Applications`.
3. Запустите приложение из папки «Программы». Значок появится в строке меню, а в Dock приложение отображаться не будет.
4. Если macOS блокирует первый запуск, нажмите по приложению правой кнопкой мыши, выберите «Открыть» и подтвердите запуск. Либо откройте «Системные настройки» → «Конфиденциальность и безопасность» → «Всё равно открыть».
5. Выдайте приложению запрошенные разрешения macOS.

Для обновления откройте новый DMG и замените прежнее приложение в папке «Программы». Не переименовывайте приложение и не меняйте путь `/Applications/Live Wallpapers for Mac.app`: это необходимо для сохранения разрешений macOS.

## Installation in English

1. Download `Live.Wallpapers.for.Mac-0.1.0.dmg` from **Assets** below.
2. Open the DMG and drag `Live Wallpapers for Mac.app` onto the `Applications` shortcut.
3. Launch it from Applications. The app appears in the menu bar and stays out of the Dock.
4. If macOS blocks the first launch, Control-click the app, choose Open and confirm. Alternatively, use System Settings → Privacy & Security → Open Anyway.
5. Grant the requested macOS permissions.

To update, open the new DMG and replace the existing app in Applications. Do not rename it or change `/Applications/Live Wallpapers for Mac.app`; the stable identity and path are required to preserve macOS permissions.

## Release identity

- Bundle ID: `com.medusa411.LiveWallpapersForMac`
- Executable: `Live Wallpapers for Mac`
- Install path: `/Applications/Live Wallpapers for Mac.app`
- Minimum system: macOS 13
