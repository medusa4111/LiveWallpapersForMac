# Live Wallpapers for Mac 0.1.1

## Что изменилось

- «Проверить обновления…» перенесено в основное меню непосредственно над «Открыть настройки…».
- Дублирующий пункт проверки обновлений удалён из меню «Сервис».

## Обновление

1. В разделе **Assets** скачайте `Live.Wallpapers.for.Mac-0.1.1.dmg`.
2. Откройте DMG и перетащите `Live Wallpapers for Mac.app` на ярлык `Applications`.
3. Подтвердите замену предыдущей версии.

Не переименовывайте приложение и не меняйте путь `/Applications/Live Wallpapers for Mac.app`: постоянные Bundle ID, executable и подпись нужны для сохранения разрешений macOS.

## Changes

- Moved Check for Updates to the main menu, directly above Open Settings.
- Removed the duplicate update action from the Service submenu.

## Updating

Download `Live.Wallpapers.for.Mac-0.1.1.dmg` from **Assets**, open it, and drag the app onto the Applications shortcut. Confirm replacement of the previous version.

Keep the app name and `/Applications/Live Wallpapers for Mac.app` path unchanged to preserve macOS permissions.
